Git

  • git init - создание репозитория
  • git clone <ссылка с GitHub> - создание копии удаленного репозитория с GitHub на локальном компьютере
  • git status - отображение состояния репозитория - какие файлы в каком состоянии находятся
  • git add <название файла> - добавление файла в отслеживаемые
  • git commit -m "<комментарий>" - сохранение изменений в репозитории
  • git push - внесение изменений на удаленный репозиторий
  • git log - отображение всех коммитов
  • git pull - получение изменений из удаленного репозитория в локальный
  • git branch <название ветки> - создание новой ветки в локальном репозитории
  • git checkout <название ветки> - переключение между ветками
